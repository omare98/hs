## USTRA HR 차트 공통함수

### Javascript

1. chart-utils.js
   * 차트공통 유틸함수 ChartUtils 추가
       * CHART_TYPE : 차트유형
       * chart() : 차트 렌더링 함수
           * id : 차트 element id (필수)
           * type : 차트 유형 (CHART_TYPE)
           * data : 차트 data
           * options : 차트 설정 옵션 (필수X)
        
           ```js
           ChartUtils.chart({
               id : '차트 element id',
               type : ChartUtils.CHART_TYPE.BAR,
               data : [],
               options : {}
           });
           ```
